⚡ 빠름

Bun 런타임 기반으로 초고속 실행

🏗️ MVC

CodeIgniter 3 스타일의 친숙한 MVC 패턴

🗄️ SQL

Bun SQL 통합 (SQLite / PostgreSQL / MySQL)

🎨 Template

PHP 스타일 템플릿 + 슬롯 시스템으로 서버 사이드 렌더링

빠른 시작

# 스캐폴딩으로 MVC 전체 생성
bun run igniter make:scaffold post --fields=title:string,content:text

# 컨트롤러만 생성
bun run igniter make:controller users --resource

# 마이그레이션 실행
bun run igniter migrate

# 개발 서버 실행
bun run dev

CLI 명령어

명령어 설명
make:scaffold <name> Controller + Model + View + Migration 전체 생성
make:controller <name> 컨트롤러 생성
make:model <name> 모델 생성
make:view <path> 뷰 템플릿 생성
make:migration <name> 마이그레이션 생성
make:middleware <name> 미들웨어 생성
make:helper <name> 헬퍼 파일 생성
make:library <name> 라이브러리 클래스 생성
migrate 마이그레이션 실행
list:routes 라우트 목록 출력
